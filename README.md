# HeApp

Health platform for efficient communication between caregivers and patients. Built at WhatTheHack: HealthTech hackathon.

## The problem

Caregivers face tight agendas and a high risk of burnout. HeApp aims to reduce the load by centralizing communication and reducing unnecessary phone calls.

## Features

- Centralized caregiver–patient messaging
- Prescription requests
- Chatbot interaction

## Repository structure

| Folder | Contents |

- **Backend:** Node.js, Express, MongoDB
- **Mobile:** App Inventor (mockup)
- **Other:** VoIP
